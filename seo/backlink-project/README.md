# Backlink Partnership Project — Blue Logic Water

**Goal:** Increase organic SEO by earning high-quality backlinks from high-authority,
topically-adjacent Utah businesses through *value-first content partnerships* — the
opposite of the bought/spam link burst we already disavowed (`../disavow-bluelogicwater.txt`).

**Method (Strategy 1 — "Adjacent Niche" Partnership Outreach):** Build genuinely useful
content assets that non-competing local businesses *want* to reference for their own
audience, then reciprocate with a feature / guest piece / resource listing. Two-sided
value exchange, not link begging.
